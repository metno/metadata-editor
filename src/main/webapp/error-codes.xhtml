Metadata editor - Error codes

Error Codes

This page gives some explanations to the error codes returned by the metadata editor in case of errors. Keep in mind that some of the error codes are quite broad, so it is important to also take a look at the actual error codes.

100

This is a general error code for errors that are not classified further. It is mostly used for exceptions that "should not happen". When these types of errors occur it is best to consult the log to fix the problem.

101

This error code is used for the cases where the editor failed to parse the template for a format correctly. The most likely cause for this is invalid XML in the template file.

102

This error code is used for all types of input/output errors that can occur. Possible causes are that the editor cannot read the requested resource for some reason or that it cannot write the content of the editor to the file. Since this category is so broad it is often necessary to consult the logs when it happens.

103

This error code is used when the editor cannot parse the metadata file it is given. This is most often due to invalid XML in the metadata file.

104

This error code is used when the editor configuration cannot be read correctly. This can either be due to invalid XML in the editor configuration XML for a format or that the widget types are invalid. I.e. that the attribute xsi:type is set to an invalid widget name.

105

This error code happens when the user asks to view a project that has not been configured in the property file for the editor.

106

This error code is used when the template file is missing for a format and the user tries to edit a file in the format.

107

This error code is used when the editor config file is missing for a format and the user tries to edit a file in the format.

108

This error code is used when the user tries to edit a metadata file, but the format that is used has not been configured in the setup.xml file for the project.

109

This error code is used when the metadata resource that is requested cannot be found. This is most likely due to requesting a file that is not present in the configured datastore for the project.

110

This error code is used when the setup.xml file for a project can be parsed correctly. This is most likely due to not following the specified format for the setup.xml file.

201

This error code is used if an attribute value for an editor variable tag in the template does not have a valid value. The problem with the attribute is given in error message.

202

This error code is used if an editor variable tag is not known to the template parser. This is most likely due to a typo in the editor template.

203

This error code is used if an editor variable is missing any of the associated $<var> strings inside the variable. To fix the problem make sure that all $<var> for the variable type is used and do not containt typos.

301

This error code is used when a variable is used in the editor configuration for a format, but the variable is not found in the template for the format. This is often due to a typo or that the variable has not been added to the template as intended

302

This error code is used when a widget configuration is not valid according to the rule for the widget. See the error message for more details on what is wrong.
